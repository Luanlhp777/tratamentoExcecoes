# ⚠️ Tratamento de Exceções em JavaScript

Projeto desenvolvido para prática de lógica de programação utilizando JavaScript, com foco em tratamento de exceções, validação de dados e controle de erros em aplicações.

O objetivo deste repositório é fortalecer o entendimento sobre captura e tratamento de erros, aplicando boas práticas de programação para tornar o código mais seguro, organizado e confiável.
